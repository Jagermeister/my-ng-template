# My Angular Template

Angular 8

## Features
- Root component template
    - Header
    - Router Outlet
    - Footer
- Routing (with Modules)
- Disclosure pop-up
- Material Modules
    - Icons (including SVG)
    - Theming

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
